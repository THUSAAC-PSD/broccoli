# Broccoli

Broccoli 是一个竞赛评测平台。题目、比赛、提交、评测等核心功能由服务端提供，其余能力通过**插件**扩展，插件可以同时增强后端与前端界面。

本文档介绍平台的各个插件及其使用方式。建议从这里开始：

- **[打印](./plugins/printing.md)** —— 面向 ICPC 风格比赛的按需代码打印，配有工作人员队列与原生打印站。

:::note[语言]

本文档提供 English 与简体中文两个版本，可通过顶部导航栏的语言选择器切换。

:::
